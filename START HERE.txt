  -------- IF YOU'RE NEW HERE: JUST DO STEP 1 ------------

  1. Double-click   Install FreeGSTBill.bat

     That's it. The installer takes care of:
       *  installing Node.js (if your PC doesn't have it)
       *  installing the app
       *  putting a "Free GST Billing Software" icon
          on your Desktop and in the Start Menu
       *  starting the server when you log in to Windows

     Allowed time: about 1-2 minutes on the first run.
     The app opens in your browser automatically when done.

     The installer does NOT need admin rights and does NOT
     write anything to Program Files or the Windows registry
     (HKLM). Everything is installed under your user profile.

  ---------------------------------------------------------


  -------- IF WINDOWS SHOWS A SECURITY WARNING ------------

  This is normal for free open-source apps that aren't code
  signed (code-signing certificates cost ~$300/year, which
  defeats the "free" part). Your data is safe.

  "Windows protected your PC" blue screen?
     -> Click  "More info"  then  "Run anyway"

  "This file came from another computer and might be blocked"?
     -> Right-click  Install FreeGSTBill.bat  ->  Properties
     -> Tick the  Unblock  box at the bottom  ->  OK
     -> Then double-click again.

  Antivirus flagged the .bat file?
     -> The script only:
          a) checks if Node.js is installed
          b) opens nodejs.org if it isn't (no auto-download)
          c) runs "npm install" to fetch app dependencies
          d) creates desktop / start-menu shortcuts
        These are all standard installer actions, but heuristic
        antivirus sometimes flags any script that creates
        shortcuts. You can read every line of the .bat file
        in Notepad before running it.

  Full source code:
     github.com/IamRamgarhia/Free-GST-Billing-Software

  -------- FIRST-TIME SETUP -------------------------------

  When the app opens for the first time, a 4-step wizard
  walks you through:

    Step 1.  Welcome  - choose 'India', 'Outside India',
                        or 'Both'. This adapts every menu
                        and label to your country.
    Step 2.  Business details - business name, address,
                                tax ID (GSTIN / VAT / TRN
                                etc. depending on country).
    Step 3.  Bank & UPI / SWIFT - what to print on invoices
                                  so clients know where to
                                  pay you.
    Step 4.  Done - start billing.

  You can change any of this later in Settings.

  ---------------------------------------------------------


  -------- WHERE IS MY DATA? ------------------------------

  PDFs of every invoice:
     "Saved Invoices" folder, sorted by client and month.

  Everything else (bills, clients, inventory, settings):
     "data" folder, plain JSON files. Yours, on your PC.

  Want to back up?
     Settings > Export Data  saves ONE file with everything.
     Save it to a USB drive, OneDrive, Google Drive,
     or email it to yourself.

  Moving to a new computer?
     See USER_GUIDE.md - section "Moving to a new computer".

  -------- HOW TO USE: STEP-BY-STEP -----------------------

  Make your first invoice:
     1. Click  +  New Invoice  in the top bar.
     2. Choose the type:
            Tax Invoice  /  Proforma  /  Credit Note
            Bill of Supply  /  Delivery Challan
     3. Type the client name (or pick a saved one).
     4. Add line items - description, qty, unit
        (kg, ltr, mtr, hrs, pcs, or your own custom unit),
        rate, tax %.
     5. Click Download PDF.  The invoice is saved
        and a PDF copy lands in the Saved Invoices folder.

  When does an invoice get saved?
     The app shows a small "All changes saved" badge
     at the top while you work.

       Draft only - not saved yet  -> Need a client name
                                      AND at least one
                                      filled-in item.
       Saving...                   -> Persisting now.
       All changes saved           -> Safe to close.

     If you click Back with unsaved changes, the app
     asks you first. Nothing is silently lost.

  -------- BEST TIP: INSTALL AS A DESKTOP APP -------------

  After the app first opens in your browser:
     1. Look for the small "Install App" icon in the URL

     2. Click Install. The app now runs in its own window
        like any other Windows app, even when offline.

  -------- TROUBLESHOOTING --------------------------------

  Page won't load (localhost:3001 says "Cannot reach this
  page")?
     -> Double-click "Free GST Billing Software" on Desktop.
        That starts the server. Then refresh the page.

  Installer says "Node.js install failed"?
     -> Install it manually from https://nodejs.org
        (pick the LTS version).
        Then run Install FreeGSTBill.bat again.

  Something else is wrong?
     -> Read USER_GUIDE.md (in this folder) - has detailed
        answers to common problems.
